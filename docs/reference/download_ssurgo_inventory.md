# Download a zipped directory containing a shapefile of the SSURGO study areas.

Download a zipped directory containing a shapefile of the SSURGO study
areas.

## Usage

``` r
download_ssurgo_inventory(raw.dir, ...)
```

## Arguments

- raw.dir:

  A character string indicating where raw downloaded files should be
  put.

## Value

A character string representing the full local path of the SSURGO study
areas zipped directory.
