# Unwraps a matrix and only keep the first n elements.

A function that unwraps a matrix and only keeps the first n elements n
can be either a constant (in which case it will be repeated), or a
vector

## Usage

``` r
unwrap_rows(mat, n)
```

## Arguments

- mat:

  A matrix

- n:

  A numeric vector

## Value

A logical vector of the same length as x
